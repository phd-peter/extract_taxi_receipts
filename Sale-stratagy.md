## 📊 단계별 판매 전략

### 🥉 **Level 1: 수동 대행 서비스**
**현재 수준 - 가장 빠른 시작**
- **서비스**: 고객이 영수증 사진을 보내면 → CSV 파일로 가공해서 전달
- **타겟**: 소규모 회사, 개인 프리랜서
- **가격**: 장당 500-1000원 또는 월 정액제 (50장 기준 2만원)
- **장점**: 초기 투자 없이 바로 시작 가능
- **단점**: 확장성 한계, 노동집약적

### 🥈 **Level 2: 웹 기반 셀프 서비스**
**반자동화 - 확장 가능한 모델**

**기술 구현:**
```python
# Flask/Django 웹앱 구조
- 파일 업로드 인터페이스
- 실시간 처리 상태 표시
- CSV 다운로드 기능
- 사용자 계정 관리
```

**서비스 특징:**
- **웹사이트**: 사용자가 직접 사진 업로드 → 자동 처리 → CSV 다운로드
- **타겟**: 중소기업, 스타트업, 회계사무소
- **가격 모델**: 
  - 프리미엄: 월 10장 무료 + 추가 장당 300원
  - 비즈니스: 월 100장 3만원
  - 엔터프라이즈: 월 500장 10만원
- **추가 기능**: 
  - 월별/연도별 리포트 생성
  - 엑셀 export 지원
  - 영수증 이미지 보관함

- **타겟**: 
  - 경비관리 솔루션 업체
  - 회계 프로그램 개발사
  - 대기업 IT 부서
- **가격**: API 호출당 과금 (건당 100-200원)

### 🏆 **Level 4: 엔터프라이즈 솔루션**

- **커스터마이징**: 고객사 요구사항에 맞춤 개발
- **타겟**: 대기업, 공공기관
- **가격**: 초기 설치비 500만원 + 연간 유지보수비 200만원
